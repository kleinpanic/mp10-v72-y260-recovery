# Notice

This repository is an unofficial recovery package.

The patch scripts, manifest, README, generated analysis notes, and release notes are original recovery tooling. The `update.bin` file is a patched derivative of the public full-size Monoprice MP10 v72 firmware image. Files under `src/vendor-reference/` are an extracted copy of the Monoprice3DP `MiniV2DeltaSourceCode` C/C++ source archive. Files under `src/official-v72/` are generated Ghidra decompiler/disassembly output from the official binary, not original vendor source. The upstream firmware repository carries GPLv3 licensing, so this repository includes the GPLv3 `LICENSE` file.

No ownership or vendor affiliation is claimed. Monoprice and Malyan names are used only to identify the affected printer and upstream firmware family.

Use at your own risk. Firmware flashing can render hardware unusable if the wrong image is used or the update process is interrupted.
