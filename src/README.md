# Source And Generated Analysis

This directory contains two different source-adjacent artifacts:

- `vendor-reference/` contains the available Monoprice-published MP10-family C/C++ source archive and extracted source tree.
- `official-v72/` contains generated analysis output from the official full-size v72 firmware binary.

## Vendor Source Reference

`vendor-reference/Monoprice3DP-MiniV2DeltaSourceCode/` is copied from the Monoprice3DP `MiniV2DeltaSourceCode` repository evidence captured for this recovery project.

Captured provenance:

- Upstream repository: `git@github.com:Monoprice3DP/MiniV2DeltaSourceCode.git`
- Upstream commit: `e800c654117d51b0070820aad61132eb60b2ffeb`
- Source archive: `MM32-F401-MARLIN-V2-HAL-5844f073761aac2619543da0502dfc4524c2a1d5.zip`
- Archive SHA-256: `157338cd85d7858ad13ffcd51cb04d45c533448688c1258ec4aebaec24e01ceb`
- Archive path: `vendor-reference/Monoprice3DP-MiniV2DeltaSourceCode/archives/`
- Extracted path: `vendor-reference/Monoprice3DP-MiniV2DeltaSourceCode/extracted/MM32-F401-MARLIN-V2-HAL-5844f073761aac2619543da0502dfc4524c2a1d5/MM32-F401-MARLIN-V2-HAL-5844f073761aac2619543da0502dfc4524c2a1d5/`
- Extracted files: 1,107 total, including 1,080 C/C++/header-style files.

This is original vendor C/C++ source, not decompiler output. Its main `Marlin/Configuration.h` contains a full-size MA10 block with `X_BED_SIZE 300`, `Y_BED_SIZE 300`, and `Z_MAX_POS 400`, plus smaller shared/default/example configurations. It is still not a proven exact source-build input for the full-size MP10 v72 binary because `Marlin/inc/version.h` reports build `66`, while the official patched artifact is v72.

## Official V72 Generated Analysis

- `official-v72/` is generated by `tools/decompile_official_v72.py`.
- Generated files are C-like pseudocode and disassembly produced by Ghidra.
- Generated files are not original vendor source and should not be treated as a source rebuild path.

The actual reproducible firmware artifact is built by `tools/create_update_bin.py`, which validates the official v72 binary hash and applies the documented Y260 binary patch.
