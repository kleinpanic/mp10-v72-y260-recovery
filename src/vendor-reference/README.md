## Scope

This is original vendor source, not Ghidra output. It is included for auditability, GPL source availability, and future reconstruction work.

The main `Marlin/Configuration.h` in this archive contains a full-size MA10 block with `X_BED_SIZE 300`, `Y_BED_SIZE 300`, and `Z_MAX_POS 400`, plus smaller shared/default/example configurations. The same tree reports `SHORT_BUILD_VERSION "66"`, so the patched firmware artifact in this repo is still generated by validating and patching the official full-size MP10 v72 binary, not by rebuilding this source tree.
